# jb-textarea
simple textarea web component
## usage
```cmd
npm i jb-textarea
```
```html
<jb-textarea label="توضیحات" value="" message="متن زیر باکس"></jb-textarea>
```
## get and set value

```js
document.querySelector("jb-textarea").value;
// retun inputed text
document.querySelector("jb-textarea").value = "salam";
set value to salam
```
